CSS Block & Inline

Block(khối) là các phần tử HTML khi tạo ra nó sở hữu 1 hàng riêng biệt. Có thể tùy chỉnh kích thước của block. VD: các thẻ div, h1, h1, p,ul, ...

Inline(nội dòng) là các phần tử HTML khi tạo ra nó có thể đứng trên cùng 1 hàng. Không thể tùy chỉnh.VD: thẻ span, strong,a,...
