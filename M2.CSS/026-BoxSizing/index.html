CSS Boxsizing

Box-sizing: content-box: thuộc tính width trong css là độ rộng của content

Box-sizing: content-box: thuộc tính height trong css là độ cao của content

Border-box: Thuộc tính width trong css là tổng độ rộng của box model

Border-box: Thuộc tính height trong css là tổng độ cao của box model
