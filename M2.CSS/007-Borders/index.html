Border

Border style dotted(kiểu các dấu chấm)

Border style dashed(kiểu các dấu gạch đứt quãng)

Border style solid(kiểu các đường liền gạch)

Border style double(kiểu có 2 đường viền chạy song song)

Border style groove(kiểu 3D)

Border style ridge(kiểu 3D mỏng)

Border style inset(kiểu 3D đổ bóng bên trong )

Border style outset(kiểu 3D đổ bóng bên ngoài )

Border style none(kiểu không có đường viền )

Border style hidden(kiểu có đường viền nhưng ẩn )

Border style mix(kiểu nhiều loại đường viền )
