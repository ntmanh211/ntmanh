Vòng lặp là quá trình lặp đi lặp lại có bước bắt đầu và kết thúc
    và có sự thay đổi sau mỗi vòng lặp
    1- for
    2- while
    3- do while
    Cú pháp vòng lặp for:
    for(bước khởi đầu; điều kiện vòng lặp; sự thay đổi sau mỗi vòng lặp){
        //code thực thi trong vòng lặp
    }
    Cú pháp vòng lặp while:
    bước khởi đầu
    while(điều kiện vòng lặp chạy){
        // code thực thi vòng lặp
        sự thay đổi sau mỗi vòng lặp
    }
